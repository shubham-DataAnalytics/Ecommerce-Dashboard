# E‑Commerce Performance Dashboard

## 📊 Overview
This dashboard provides a comprehensive analysis of e‑commerce performance across revenue, profit, and quantity sold. It highlights key insights by category, subcategory, customer, payment mode, and geography, enabling data-driven decisions for growth and optimization.

## 🔨 Process
1. **Data Modeling**  
   - Structured raw sales data with relationships between orders, products, customers, and regions.  
   - Enabled filtering by state and order date range (2018).

2. **DAX Measures**  
   - Created metrics for total revenue (₹438K), profit (₹37K), and quantity sold (5615 units).  
   - Built logic for profit breakdowns by category, subcategory, customer, and payment mode.

3. **Visualizations**  
   - Bar charts for profit by category, subcategory, customer, and city.  
   - Donut chart for payment mode distribution.  
   - Treemap for state-wise profit contribution.  
   - Interactive filters for state selection and date range.

## 🎯 Key Insights
- Top categories: Clothing, Electronics, and Furniture.  
- Highest subcategory profits: Printers, Bookcases, Sarees.  
- Leading customers: Aarushi, Madan, Shrichand.  
- Credit Card and COD dominate payment modes (~34% each).  
- Madhya Pradesh and Maharashtra lead in state-wise profit.
